## Usage

```sh
nvmx install 8.9.4   # Install specific version
nvmx install 8       # Install the latest version of 8.x.x
nvmx install 3.3.1   # Install io.js 3.1.1

nvmx install lts     # Install latest LTS

nvmx install nightly # Install latest nightly

nvmx install --chakracore 8.9.4   # Install Node.js ChakraCore
nvmx install --chakracore nightly # Install latest nightly of Node.js ChakraCore
```
